# GreenDZ

🔗 [Try it here](https://green-dz-app.netlify.app/)

GreenDZ is an Arabic (RTL) recycling web app for Algeria. It helps people find and add recycling points on an interactive map, learn how to sort waste, and discover local environmental events.

## Features
- Interactive map (OpenStreetMap / Leaflet, no API key required) with recycling points by wilaya and waste type
- Add-a-point form with location picker
- Waste-type filters (plastic, paper, glass, metal, e-waste)
- Learn section with sorting tips
- Local environmental events listing
- Fully responsive, RTL Arabic layout

## Tech Stack
React, TypeScript, Tailwind CSS, react-leaflet + OpenStreetMap
